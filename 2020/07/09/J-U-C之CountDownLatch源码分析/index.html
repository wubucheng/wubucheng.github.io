概念与作用

CountDownLatch是JUC包下提供的一个工具类，它的作用是让一个或者一组线程等待其他线程执行完成后，自己再接着执行，count数不为0则线程进行等待。

API

  • await：执行等待，计数不为0则进入等待
  • await(long timeout, TimeUnit unit)：超时则自动唤醒，继续往下走
  • coutDown：计数减一
  • getCount：获取当前计数的值
